# 🏥 Hospital Management System SQL Project

A complete Hospital Management System database project built using **PostgreSQL**. This project demonstrates SQL skills including database design, data import, data analysis, and advanced SQL queries using a real-world healthcare dataset.

---

## 📌 Project Overview

This project simulates a Hospital Management System where data related to patients, doctors, appointments, diagnoses, treatments, billing, and hospital operations are stored and analyzed.

The objective is to practice SQL concepts from beginner to advanced while solving real-world business problems.

## 📥 Importing Dataset

1. Create the database.

```sql
CREATE DATABASE hospital_management;
```

2. Run the table creation script.

3. Import the CSV using pgAdmin Import Tool.

4. Execute all SQL queries.
